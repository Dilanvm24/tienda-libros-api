# 📚 API Tienda de Libros

Esta es una API REST profesional para la gestión de una librería, incluyendo autenticación de usuarios y documentación automática.

## 🚀 Tecnologías utilizadas
* **Node.js & Express**: Servidor y rutas.
* **MongoDB & Mongoose**: Base de datos NoSQL.
* **JWT (JSON Web Tokens)**: Seguridad y sesiones.
* **Bcrypt**: Encriptación de contraseñas.
* **Swagger**: Documentación interactiva.

## 🛠️ Instalación y Uso

1. Clona el repositorio.
2. Ejecuta `npm install` para instalar las dependencias.
3. Crea un archivo `.env` basado en la configuración del servidor (Mongo URI, Port, JWT Secret).
4. Inicia el servidor con:
   ```bash
   node server.js
